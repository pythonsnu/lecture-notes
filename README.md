## 실습 자료

아래 실습 자료의 링크를 클릭해서 파일에 연결한 후,
1. 메뉴 중 **파일**을 선택한 후
1. **실습 모드에서 열기**를 클릭하면 됩니다.

실습 자료를 자신의 구글 드라이브에 저장하고 싶으면,
1. 메뉴 중 **파일**을 선택한 후
1. **드라이브에 사본 저장**을 클릭하면 됩니다.
